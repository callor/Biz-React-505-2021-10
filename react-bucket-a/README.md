- SPA(Single Page App)로 구현하는 나의 버킷 리스트
- React를 사용하여 화면구현
- Nodejs + MongoDB를 연동한 서버구현

## CRUD

- 입력창에 버킷 입력후 Enter 누르면 바로 insert
- FLAG 를 클릭하면 중요, 매우중요, 긴급, 일반 이 번갈아 표시
- 완료항목 클릭하면 완료일자가 등록
- 취소를 체크하면 취소선이 나타나고
- 항목 클릭하면 업데이트를 수행할수 있도록
